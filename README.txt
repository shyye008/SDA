R function.R: R functions for the proposed method. 
	SDA: R function for single hypothesis testing
	SDA_knock: R function for multiple hypothesis testing with FDR control

Simulation codes: the folder contains R scripts used to generate the .csv files in the "Outputs" folder.
	Sim_slice.R: simulation for Section 4.2.1, generate "out_slice.csv" and "out_slice.csv"
	Sim_rate1.R: simulation for Section 4.2.2, SDA-CvM and SDA-KS, generate "out_single1.csv", "out_single2.csv", "out_multi1.csv", and "out_multi2.csv"
	Sim_rate2.R: simulation for Section 4.2.2, SDA-Chi, generate "out_single1_chi.csv", "out_single2_chi.csv", "out_multi1_chi.csv", and "out_multi2_chi.csv"
	Sim_rate2.R: simulation for Section 4.2.2, SDA-Chi, generate "out_single_part_1.csv"-"out_single_part_8.csv" and "out_multi_part_1.csv"-"out_multi_part_8.csv"
	Sim_MH.R: simulation for Section 4.2.3, generate "out_FDR.csv"
	Sim_miss1.R: simulation for Section 4.2.4, empirical selection rates, generate "out_miss.csv"
	Sim_miss2.R: simulation for Section 4.2.4, multiple hypothesis testing, generate "out_miss_MH1.csv"-"out_miss_MH4.csv"
	Sim_miss3.R: simulation for Section 4.2.5, generate "out_miss_LASSO.csv", "out_miss_SIS.csv", and "out_miss_LS.csv"


Outputs: the folder contains the .csv files generated by the R scripts in the "Simulation codes" folder (see above), as well as R codes used to reproduce the tables and figures in the paper.
	Output.R: R codes used to reproduce the tables and figures in the paper.
		Line 1-262: read "out_slice.csv" and "out_slice.csv", output Figures 1, S1, and S2
		Line 264-358: read "out_single1.csv", "out_single2.csv", "out_multi1.csv", and "out_multi2.csv", output results for SDA-KS and SDA-CvM in Tables 1 and S2
		Line 360-445: read "out_single1_chi.csv", "out_single2_chi.csv", "out_multi1_chi.csv", and "out_multi2_chi.csv", output results for SDA-Chi in Tables 1 and S2
		Line 447-484: read "out_single_part_1.csv"-"out_single_part_8.csv" and read "out_multi_part_1.csv"-"out_multi_part_8.csv", output results for HP in Tables 1 and S2
		Line 486-560: read "out_FDR.csv", output Figure S3
		Line 562-625: read "out_miss.csv", output results for Table 2
		Line 627-720: read "out_miss_MH1.csv"-"out_miss_MH4.csv", output Figure S4
		Line 722-1106: read "out_miss_LASSO.csv", "out_miss_SIS.csv", and "out_miss_LS.csv", output Figures 2 and S5
